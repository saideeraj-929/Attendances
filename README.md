# 🏫 Attendance Management System 📝

A lightweight, desktop-based GUI application built using Python and **Tkinter** to track, manage, and save student attendance records with ease. ✨

---

## 🚀 Features

* **➕ Add Records:** Quickly mark a student as **Present** or **Absent**.
* **📊 Listbox View:** Instantly view a clean list of marked attendance.
* **💾 Persistent Storage:** Automatically saves attendance records to a local text file (`attendance.txt`).
* **🔄 Auto-Load:** Automatically retrieves and displays previously saved attendance on startup.
* **🧹 Clear Functionality:** Reset and clear the current active view with a single click.
* **🎨 Modern GUI Design:** An intuitive, pastel-themed interface with vibrant, color-coded buttons.

---

## 🛠️ Prerequisites

To run this application, you only need **Python 3.x** installed on your system. 🐍 
*(Tkinter is included by default in standard Python installations!)*

---

## ⚙️ How to Run

1. **📥 Clone or Download** this repository to your local machine.
2. Save the Python script as `main.py`.
3. Open your terminal or command prompt in the project directory and run:

```bash
python main.py
